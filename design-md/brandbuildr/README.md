# BrandBuildr Design System

[DESIGN.md](./DESIGN.md) for the BrandBuildr strategic brand-building platform. This documents the product visual language: calm professional surfaces, phase-driven colour, tonal depth (not shadows on cards), and a dark slate canvas for session and deploy moments. It is not an official public marketing extract; it is a contributor-authored system for AI-aligned UI work.

**Upstream:** this folder is not part of the [VoltAgent/awesome-design-md](https://github.com/VoltAgent/awesome-design-md) collection on GitHub. It lives only in this local repository (or your fork) until you choose to publish it elsewhere.

## Files

| File | Description |
|------|-------------|
| `DESIGN.md` | Complete design system documentation (YAML tokens + sections) |
| `preview.html` | Interactive design token catalog (light — standard app surfaces) |
| `preview-dark.html` | Interactive design token catalog (dark slate canvas) |

Use [DESIGN.md](./DESIGN.md) as a reference for AI agents (Claude, Cursor, Stitch) to generate UI that matches the BrandBuildr design language.
